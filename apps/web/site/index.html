SHA-256d Proof-of-Work

Double-SHA256 mining with automatic difficulty adjustment every 2,016 blocks.

🔗

UTXO Transaction Model

Clean unspent-output model with P2WPKH addresses, Schnorr signatures, and BIP-44 HD wallets.

🪙

Sound Money

Hard-capped at 21 million KUBER with a predictable halving schedule — no inflation surprises.

⚡

Full Node in 6 MB

Entire node compiled to a single optimized Rust binary. Run on any machine — no runtime dependencies.

📡

Bitcoin-Compatible RPC

Standard JSON-RPC 2.0 interface: getblock, sendrawtransaction, getblockchaininfo, and more.

📦

Complete Stack

Node, web wallet, block explorer, monitoring (Prometheus + Grafana), Docker, Kubernetes — all included.

Quick Start

Run a node, create a wallet, and send your first transaction in minutes. 

# Download and run a full node
$ kubercoin node

# Create a wallet
$ kubercoin new-wallet my_wallet.json
Address: kuber1qw508d6qejxtdg4y5r3zarvary0c5xw7k

# Start mining
$ kubercoin mine-to kuber1qw508d6qejxtdg4y5r3zarvary0c5xw7k 10

# Check balance
$ kubercoin get-balance my_wallet.json
Total: 500.00000000 KUBER (50 per block × 10 blocks)

# Send a payment
$ kubercoin send my_wallet.json kuber1recipient... 25.0

# Or run the full stack with Docker
$ docker compose up -d
