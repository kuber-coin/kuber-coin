## 2. Data We Collect

### 2.1 Blockchain Data (Public & Permanent)

The Kubercoin blockchain records **all transactions publicly**:

**What's Stored:**
- Transaction amounts and fees
- Sender and recipient addresses
- Transaction timestamps
- Block data and merkle trees
- Smart contract interactions (future)

**Privacy Features:**
- Optional transaction obfuscation (Dandelion++)
- Optional ring signatures (future)
- Optional zero-knowledge proofs (future)
- Privacy is **opt-in**, not mandatory

**Cannot Be Deleted:**
- Blockchain data is **immutable** and **permanent**
- Data is replicated across thousands of nodes
- Even we cannot delete or modify transactions
- "Right to erasure" does not apply to blockchain data

**Pseudonymity, Not Anonymity:**
- Addresses are pseudonymous (not linked to real identities)
- With enough data, addresses **can be de-anonymized**
- Reusing addresses reduces privacy
- Best practice: Use new address for each transaction

### 2.2 Node Software (Local Only)

When you run a Kubercoin node, data is stored **locally on your device**:

**What's Stored Locally:**
- Blockchain data (blocks and transactions)
- UTXO set (unspent outputs)
- Peer connections and IP addresses
- Wallet data (if using built-in wallet)
- Configuration files
- Log files (optional)

**Not Sent to Us:**
- We **do not receive** this data
- No telemetry or analytics by default
- No phone-home functionality
- No automatic crash reports
- No usage statistics

**Your Responsibility:**
- Secure your own data
- Encrypt wallet files
- Backup seed phrases
- Protect private keys

### 2.3 Website Analytics (Optional)

Our websites may use **minimal analytics**:

**What We May Collect:**
- Page views and navigation

**What We Don't Collect:**

**How to Opt Out:**

## 13. Specific Scenarios

### 13.1 Using Tor or VPN

**Tor:**
- Hides your IP address from websites
- Does **not** hide blockchain transactions
- Transactions are still public
- Node connections may be limited

**VPN:**
- Changes your apparent location
- Does **not** provide blockchain privacy
- Choose reputable VPN providers

### 13.2 Running a Node

**Privacy implications:**

- Your IP address is visible to connected peers
- Transactions you broadcast may be associated with your IP
- Use Tor for better IP privacy
- NAT traversal may expose your home IP

**Peer data:**
- You collect peer IP addresses locally
- Not sent to us
- Stored only for P2P networking
- Can be deleted anytime

### 13.3 Mining

**Pool mining:**
- Pool operators see your IP address
- Pool operators see your payment address
- Choose pools with privacy policies
- Solo mining is more private

**Block rewards:**
- Coinbase address is public
- All can see miner addresses
- Consider using new addresses

### 13.4 Using a Wallet

**Built-in wallet:**
- Stores keys locally on your device
- Not synced to cloud
- Your responsibility to backup
- Encrypted wallet files recommended

**Seed phrases:**
- Never share with anyone (including us!)
- Needed to recover wallet
- Write down and store securely
- We cannot recover lost seeds
